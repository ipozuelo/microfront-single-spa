Single-SPA Microfrontend

Single-spa es una biblioteca de JavaScript que permite crear aplicaciones web de una sola página (SPA) a partir de múltiples aplicaciones independientes o microfrontends. La idea central de Single-spa es que cada microfrontend puede estar desarrollado con un marco o tecnología diferente, como Angular, React, Vue.js o cualquier otra de tu elección.

Para crear un proyecto root, debes ejecutar este comando y seguir las opciones que te van mostrando.
